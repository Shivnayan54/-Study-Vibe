TROUBLESHOOTING LOGIN ISSUES
================================================================

Can't login? Check these:

❌ Error: "No admin account found with this email"
   → User not created in Firebase Console
   → Solution: Follow steps above to create user

❌ Error: "Incorrect password"
   → Password typed wrong
   → Solution: Copy-paste: #BUGGATI41

❌ Error: "This account has been disabled"
   → User disabled in Firebase Console
   → Solution: Enable user in Firebase Console

❌ Page keeps loading / not redirecting
   → Firebase not configured correctly
   → Solution: Check js/firebase-config.js
